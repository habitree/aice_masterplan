## 1. AICE란?

**AICE (AI Certification Examination)**는 KT(한국통신)에서 개발하고 한국경제신문사와 공동 주관하는 인공지능 활용능력 인증 시험으로, 국가공인 민간자격으로 지정되었습니다.

### AICE Associate 시험 개요

- **시험명**: AICE Associate (Associate Level)
- **주관**: KT(개발) + 한국경제신문사(공동 주관)
- **목적**: AI/ML 기초 지식 및 실무 능력 검증
- **난이도**: 입문~중급 수준
- **자격 유형**: 국가공인 민간자격

### 시험 특징

- **실무 중심**: 이론보다 실무 적용 능력 중시
- **코딩 필수**: Python 기반 데이터 분석 및 머신러닝 실습
- **실시간 평가**: 실습 환경에서 직접 코드 작성

---

## 2. 시험 구성

### 시험 영역

1. **Python 기초** (20%)
   - 기본 문법, 자료형, 제어문
   - 함수, 모듈, 클래스

2. **데이터 분석** (25%)
   - NumPy, Pandas 활용
   - 데이터 탐색 및 전처리

3. **시각화** (10%)
   - Matplotlib, Seaborn
   - 데이터 시각화 기법

4. **머신러닝** (30%)
   - 지도학습 (회귀, 분류)
   - 비지도학습 (클러스터링)
   - 모델 평가 및 선택

5. **실전 문제** (15%)
   - 종합 실습 문제
   - 기출 유형 분석

### 시험 형식

- **시험 시간**: 120분
- **문제 유형**: 객관식 + 실습 문제
- **실습 환경**: Jupyter Notebook 기반
