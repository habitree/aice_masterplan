## 1. 파이썬 소개

### 1.1 파이썬이란?

**파이썬(Python)**은 1991년 귀도 반 로섬(Guido van Rossum)이 개발한 고급 프로그래밍 언어입니다.

**특징:**
- **간결한 문법**: 읽기 쉽고 작성하기 쉬운 코드
- **인터프리터 언어**: 컴파일 없이 바로 실행 가능
- **다양한 라이브러리**: 데이터 분석, 웹 개발, AI/ML 등 다양한 분야 지원
- **크로스 플랫폼**: Windows, Mac, Linux 모두 지원

### 1.2 파이썬 설치 확인

```python
# Python 버전 확인
import sys
print(sys.version)

# 출력 예시: 3.9.7 (default, ...)
```

## 2. 변수와 할당

### 2.1 변수 선언

파이썬에서는 변수 선언 시 타입을 명시하지 않습니다.

```python
# 변수 선언 및 할당
name = "AICE"
age = 25
height = 175.5
is_student = True

print(name)      # AICE
print(age)       # 25
print(height)    # 175.5
print(is_student) # True
```

### 2.2 변수명 규칙

- 영문자, 숫자, 언더스코어(_) 사용 가능
- 숫자로 시작 불가
- 예약어 사용 불가 (if, for, class 등)
- 대소문자 구분

```python
# 올바른 변수명
student_name = "홍길동"
student_age = 20
_total_score = 100

# 잘못된 변수명
# 2name = "홍길동"  # 숫자로 시작 불가
# class = "A반"     # 예약어 사용 불가
```

> 💡 **Tip:** 변수명은 의미를 명확히 표현하는 것이 좋습니다. `x`, `y`보다는 `student_name`, `total_score`가 더 좋습니다.

## 3. 주석과 출력

### 3.1 주석

```python
# #-> 한 줄 주석 ,  """ -> 여러줄 편법 여러줄 주석

# 여러 줄 주석은
# 각 줄마다 # 기호를 사용합니다
# 이것이 가장 정확한 방법입니다

"""
여러 줄로 설명을 적어놓을 때
주석처럼 보이지만,
사실은 '여러 줄 문자열'입니다.
실행에 사용되지 않기 때문에
겉보기로만 주석처럼 동작합니다.

"""
'''
작은따옴표 버전도 동일합니다.
이것 역시 문자열이며,
정식 주석은 아닙니다.
'''
```

> ⚠️ **주의**: Python에는 실제로 여러 줄 주석이 없습니다. `"""`와 `'''`는 문자열 리터럴이며, 표현식의 일부가 아니면 평가되지만 결과가 없어서 무시될 뿐입니다. Jupyter 노트북에서는 이런 문자열이 실제로 실행되므로, 여러 줄 주석이 필요하면 각 줄에 `#`을 사용하는 것이 더 정확합니다.

### 3.2 출력 함수 print()

```python
# 기본 출력
print("Hello, AICE!")

# 여러 값 출력
name = "홍길동"
age = 25
print("이름:", name, "나이:", age)

# 포맷팅 (f-string, Python 3.6+)
print(f"이름: {name}, 나이: {age}")

# 포맷팅 (format 메서드)
print("이름: {}, 나이: {}".format(name, age))

# 구분자 지정
print("Python", "AICE", "Data", sep="-")  # Python-AICE-Data

# 끝 문자 지정
print("첫 번째", end=" ")
print("두 번째")  # 첫 번째 두 번째
```

### 3.3 이스케이프 문자

이스케이프 문자는 백슬래시(`\`)와 특정 문자를 조합하여 특수한 의미를 표현하는 문자입니다. 문자열 내에서 특수 문자를 출력하거나 특수한 동작을 수행할 때 사용합니다.

**📌 주요 이스케이프 문자 요약표**

| 표기 | 의미 | 예시 | 출력 결과 |
|------|------|------|----------|
| `\"` | 큰따옴표 출력 | `"이름: \"홍길동\""` | 이름: "홍길동" |
| `\'` | 작은따옴표 출력 | `'It\'s Python'` | It's Python |
| `\\` | 백슬래시 출력 | `"경로: C:\\Users"` | 경로: C:\Users |
| `\n` | 줄바꿈(New line) | `"안녕\n하세요"` | 안녕<br>하세요 |
| `\t` | 탭(Tab) | `"이름:\t홍길동"` | 이름:    홍길동 |
| `\r` | 캐리지 리턴 | `"Hello\rWorld"` | World (커서를 줄의 시작으로 이동) |
| `\b` | 백스페이스 | `"Hello\b"` | Hell (한 글자 삭제) |
| `\f` | 폼 피드 | `"Page1\fPage2"` | 페이지 나누기 (프린터용) |
| `\v` | 수직 탭 | `"Line1\vLine2"` | 수직 탭 (거의 사용 안 함) |

```python
# 이스케이프 문자 사용 예시
print("큰따옴표: \"Python\"")        # 큰따옴표: "Python"
print('작은따옴표: It\'s great')     # 작은따옴표: It's great
print("백슬래시: C:\\Users\\Desktop") # 백슬래시: C:\Users\Desktop
print("줄바꿈:\n첫 번째 줄\n두 번째 줄")
# 출력:
# 줄바꿈:
# 첫 번째 줄
# 두 번째 줄
print("탭:\t이름\t나이")            # 탭:    이름    나이
print("백스페이스: Hello\b")        # 백스페이스: Hell
```

> 💡 **Tip**: 
> - 문자열 내에서 큰따옴표를 출력하려면 `\"` 사용
> - 문자열 내에서 작은따옴표를 출력하려면 `\'` 사용 (또는 반대 따옴표 사용)
> - Windows 경로를 문자열로 표현할 때는 `\\` 사용 또는 `r"경로"` (raw string) 사용
> - 가장 많이 사용하는 이스케이프 문자: `\n` (줄바꿈), `\t` (탭)

## 4. 연산자

### 4.1 산술 연산자

```python
a = 10
b = 3

print(a + b)   # 13 (덧셈)
print(a - b)   # 7  (뺄셈)
print(a * b)   # 30 (곱셈)
print(a / b)   # 3.333... (나눗셈)
print(a // b)  # 3  (몫)
print(a % b)   # 1  (나머지)
print(a ** b)  # 1000 (거듭제곱)
```

### 4.2 비교 연산자

```python
a = 10
b = 3

print(a > b)   # True
print(a < b)   # False
print(a >= b)  # True
print(a <= b)  # False
print(a == b)  # False (같음)
print(a != b)  # True  (다름)
```

### 4.3 논리 연산자

```python
x = True
y = False

print(x and y)  # False (AND)
print(x or y)   # True  (OR)
print(not x)    # False (NOT)
```

### 4.4 할당 연산자

할당 연산자는 변수에 값을 할당하면서 동시에 연산을 수행하는 연산자입니다. 코드를 간결하게 작성할 수 있습니다.

```python
# 초기값 설정
a = 10
print(f"초기값: a = {a}")  # 초기값: a = 10

# += : 덧셈 후 할당 (a = a + 5와 동일)
a += 5   # a = a + 5
print(f"a += 5 후: a = {a}")  # a += 5 후: a = 15

# -= : 뺄셈 후 할당 (a = a - 3와 동일)
a -= 3   # a = a - 3
print(f"a -= 3 후: a = {a}")  # a -= 3 후: a = 12

# *= : 곱셈 후 할당 (a = a * 2와 동일)
a *= 2   # a = a * 2
print(f"a *= 2 후: a = {a}")  # a *= 2 후: a = 24

# /= : 나눗셈 후 할당 (a = a / 4와 동일, 결과는 실수형)
a /= 4   # a = a / 4
print(f"a /= 4 후: a = {a}")  # a /= 4 후: a = 6.0

# //= : 몫 연산 후 할당 (a = a // 2와 동일, 정수 나눗셈)
a //= 2  # a = a // 2
print(f"a //= 2 후: a = {a}")  # a //= 2 후: a = 3.0

# %= : 나머지 연산 후 할당 (a = a % 2와 동일)
a %= 2   # a = a % 2
print(f"a %= 2 후: a = {a}")  # a %= 2 후: a = 1.0
```

**할당 연산자 정리:**

| 연산자 | 의미 | 예시 | 동일한 표현 |
|--------|------|------|------------|
| `+=` | 덧셈 후 할당 | `a += 5` | `a = a + 5` |
| `-=` | 뺄셈 후 할당 | `a -= 3` | `a = a - 3` |
| `*=` | 곱셈 후 할당 | `a *= 2` | `a = a * 2` |
| `/=` | 나눗셈 후 할당 | `a /= 4` | `a = a / 4` |
| `//=` | 몫 연산 후 할당 | `a //= 2` | `a = a // 2` |
| `%=` | 나머지 연산 후 할당 | `a %= 2` | `a = a % 2` |
| `**=` | 거듭제곱 후 할당 | `a **= 3` | `a = a ** 3` |

> 💡 **Tip**: 할당 연산자는 반복문이나 카운터 변수를 다룰 때 매우 유용합니다. 예를 들어 `count += 1`은 `count = count + 1`보다 간결하고 읽기 쉽습니다.

## 5. 실습

### 실습 1: 기본 계산기

```python
# 사용자 입력 받기
num1 = float(input("첫 번째 숫자를 입력하세요: "))
num2 = float(input("두 번째 숫자를 입력하세요: "))

# 계산
sum_result = num1 + num2
diff_result = num1 - num2
prod_result = num1 * num2
quot_result = num1 / num2

# 결과 출력
print(f"\n덧셈: {num1} + {num2} = {sum_result}")
print(f"뺄셈: {num1} - {num2} = {diff_result}")
print(f"곱셈: {num1} * {num2} = {prod_result}")
print(f"나눗셈: {num1} / {num2} = {quot_result}")
```

### 실습 2: 학생 정보 출력

```python
# 학생 정보
student_name = "홍길동"
student_id = "2024001"
korean_score = 85
math_score = 92
english_score = 78

# 평균 계산
average = (korean_score + math_score + english_score) / 3

# 결과 출력
print("=" * 40)
print("학생 정보")
print("=" * 40)
print(f"이름: {student_name}")
print(f"학번: {student_id}")
print(f"국어: {korean_score}점")
print(f"수학: {math_score}점")
print(f"영어: {english_score}점")
print(f"평균: {average:.2f}점")
print("=" * 40)
```

## 6. 요약

### 핵심 정리

1. **변수**: 타입 선언 없이 바로 할당 가능
2. **주석**: `#` (한 줄 주석, 여러 줄은 각 줄마다 `#` 사용)
3. **출력**: `print()` 함수 사용, f-string으로 포맷팅
4. **연산자**: 산술, 비교, 논리, 할당 연산자 지원

### AICE 실전 팁

- **변수명**: 의미 있는 이름 사용 (코드 가독성 향상)
- **f-string**: Python 3.6+에서 가장 편리한 포맷팅 방법
- **연산자 우선순위**: 괄호를 사용하여 명확하게 표현
- **타입 확인**: `type()` 함수로 변수 타입 확인 가능

```python
# 타입 확인
x = 10
print(type(x))  # <class 'int'>

y = "Hello"
print(type(y))  # <class 'str'>
```
