## 🚀 시작하기

### 1. 진도 관리

`progress_dashboard.html` 파일을 브라우저에서 열어 학습 진도를 관리하세요.

- 체크박스로 완료 항목 표시
- 자동 진행률 계산
- 학습 시간 기록
- 일일 학습 로그 생성

### 2. 학습 방법

1. **32일_학습일정.md** 확인하여 오늘 학습 내용 파악
2. 해당 섹션의 **MD 파일**로 이론 학습
3. **HTML 파일**을 브라우저에서 열어 시각적으로 확인
4. 실습 코드 작성 및 문제 풀이
5. `progress_dashboard.html`에서 진도 업데이트

### 3. 파일 사용법

- **MD 파일**: 일반 텍스트 에디터나 마크다운 뷰어로 열기
- **HTML 파일**: 웹 브라우저에서 열기
- **템플릿 파일**: 새로운 문서 작성 시 참고

## 📖 커리큘럼

## 🔄 업데이트

각 파일은 독립적으로 업데이트 가능합니다.

- 특정 파일만 수정: 해당 파일만 재생성
- 섹션 확장: 해당 폴더에 새 파일 추가
- 템플릿 변경: templates 폴더의 파일 수정

## 📝 참고사항

- 모든 HTML 파일은 CSS가 내부에 포함되어 있어 독립적으로 사용 가능
- 블로그 업로드 시 HTML → 이미지 변환 스크립트 사용 가능
- 학습 진도는 브라우저 localStorage에 저장됨
