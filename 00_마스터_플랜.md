## 🎨 문서 스타일

### MD 파일
- GFM (GitHub Flavored Markdown) 기반
- 목차, 개념 설명, 예제 코드
- 실습 데이터 다운로드 안내
- 체크리스트, AICE 실전 팁, 요약

### HTML 파일
- 다크테마 (검은색 배경, 흰색/회색 텍스트)
- 박스형 UI (섹션 박스, 강조 박스)
- Tip/Warning/Summary 박스
- 코드블록 스타일 강화
- CSS는 문서 내부 포함
